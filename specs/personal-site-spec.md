# Personal Site Feature Specification

## Goal

Create a GitHub Pages-compatible personal site that supports a blog, online resume, and portfolio showcase.

## Users

- Recruiters and hiring managers scanning experience and impact.
- Technical leaders and peers evaluating engineering judgment.
- The site owner adding writing, resume updates, and project notes over time.

## Functional Requirements

- Home page with positioning, summary, focus areas, featured projects, and recent writing.
- About page with professional narrative.
- Resume page with experience, skills, certifications, and print-to-PDF support.
- Projects index and project detail pages sourced from Markdown content.
- Blog index and post pages sourced from Markdown content.
- RSS feed for blog posts.
- Accessible navigation, semantic structure, responsive layout, and 404 page.

## Non-Functional Requirements

- Static build suitable for GitHub Pages.
- Minimal JavaScript.
- No backend runtime.
- Clear placeholder labeling where real content is missing.
