title: "From Skills Page to Living Technical Portfolio"
description: "Why a static GitHub Pages site can become a maintainable home for writing, resume content, and project notes."
pubDate: 2026-06-28
tags: ["GitHub Pages", "Astro", "portfolio"]
---

A skills page is useful, but a living portfolio can do more. It can collect writing, project case studies, technical decisions, and resume updates in one static site that is easy to maintain.

This site uses Astro content collections for blog posts and projects. That keeps updates close to Markdown while still supporting strong structure and good build-time checks.

Future improvements can include a real resume PDF, richer project screenshots, and deeper case studies once publishable details are available.
